FAQ

How do i become a member of staff?

Head over to our applications channel and apply if applications are open

How can i join the game server?

If you are on a computer, simply use the "In-Game Server" button on the navigation menu. Otherwise, you can join the game server by using the code OregonCity and entering it in the "Join by Code" section under ER:LC's private server menu tab.

How can i appeal a ban?

To appeal a ban Click Here. You have to appeal Game and Discord bans seperately.

How do i report a staff member?

Go to the Tickets Channel and create an IA Ticket. Before doing this, ensure you have evidence and that what they did was against the rules.
